# Axis Templates

63 landing page templates for the [Axis Platform](https://github.com/brandpro1/axis-webapp).

Live previews at: **https://templates-axis.netlify.app**

## Browse & Use

Each template is a standalone HTML file at `/{slug}/index.html`.

- Brazilian Portuguese content
- Fully responsive
